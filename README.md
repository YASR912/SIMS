## 🎯 Overview

**Smart Inventory** is a modern, web-based **Inventory Management System** designed to streamline inventory operations for businesses and organizations.

The system enables efficient handling of **products, suppliers, sales, stock levels, and financial data**, along with **predictive analysis** to support data-driven decision making.  
It is developed as a **Final Year Project**, focusing on scalability, security, and real-world usability.

---

## ✨ Key Features

<details>
<summary><strong>🔐 Security & Authentication</strong></summary>

- Role-based access control (Admin, Manager, etc.)
- Secure password hashing
- Session-based authentication
- Protection against unauthorized access
</details>

<details>
<summary><strong>📦 Inventory Control</strong></summary>

- Real-time stock tracking
- Supplier and purchase order management
- Stock movement history
- Warehouse and location-based inventory tracking
- Inventory valuation and cost management
</details>

<details>
<summary><strong>💹 Sales Management</strong></summary>

- Sales transaction recording
- Automatic stock deduction on sales
- Sales reports with custom date ranges
</details>

<details>
<summary><strong>📊 Reports & Analytics</strong></summary>

- Sales, Inventory, Supplier, and Financial reports
- Date-range based filtering
- Export reports in text format
- Predictive analysis using historical data
</details>

<details>
<summary><strong>🖥️ User Interface</strong></summary>

- Clean and intuitive dashboard
- Role-specific UI views
- Report generation and printing
- Fully responsive design

 🚀 Quick Start
```bash
# Clone repository
git clone https://github.com/YASR912/SIMS.git

# Navigate to project
cd SIMS

# Database Setup Guide

## Prerequisites
- XAMPP or similar local server environment with MySQL
- phpMyAdmin access

## Setup Steps

### 1. Access phpMyAdmin
- Navigate to http://localhost/phpmyadmin in your web browser
- Login if required

### 2. Create Database
- Click "New" in the left sidebar
- Enter database name: `smart_inventory_db`
- Click "Create"

### 3. Import Database Structure
- Select `smart_inventory_db` from sidebar
- Click "Import" tab
- Browse and select `Database/smart_inventory.sql`
- Click "Go" to execute import

## Verification
- The database should now be populated with necessary tables
- You can verify by checking the structure in phpMyAdmin
```
